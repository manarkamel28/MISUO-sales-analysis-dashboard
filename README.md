## 📊 MISUO Sales Analysis Dashboard

# 📌 Project Overview

This project focuses on cleaning, transforming, analyzing, and visualizing sales data using Microsoft Excel and Power Query. The goal was to transform raw sales data into an interactive dashboard that supports business decision-making.

## 📂 Dataset

The dataset contains sales transactions, including products, customers, order status, sales channels, cities, colors, quantities, and pricing information.

## 🛠️ Tools Used

Microsoft Excel

Power Query

Pivot Tables

Pivot Charts

Slicers

Excel Dashboard

### 🧹 Data Cleaning & Transformation

The raw dataset contained several data quality issues that required preprocessing before analysis.

# Tasks Performed

✔ Removed duplicate records

✔ Handled missing values

✔ Corrected inconsistent data

✔ Standardized data formats

✔ Cleaned text values

✔ Prepared the dataset using Power Query

✔ Organized the data for Pivot Tables and dashboard creation
